# Social Media Analytics Dashboard
Social media analytics dashboard shows social media posts from different platforms.

## Dashboard feature
- User details [Name: Bio: Links:]
- Number of followers
- Number of followings
- Number of posts
- Account reached
- Total stories
- Total follows
- Total posts
- Total save
- Total comments
- Total shares
